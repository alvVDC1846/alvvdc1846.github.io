Reglas de UFW

Permitir un puerto

ufw allow 22

Permitir un puerto por TCP/UDP

ufw allow proto tcp from any to any port 80
ufw allow proto udp from any to any port 1194

Permitir un rango de IP

ufw allow from 10.8.0.0/24 to any port 443

Denegar una IP

ufw deny from 2.139.173.60

Transferencias

Sincronización

rsync -r -v -h -e ssh folder/ user@ip:~/dest_folder/ --progress --stats --exclude dir

-r recurre a directorios y subdirectorios

-v verbose

-h unidades de tamaño comprensibles

-e ssh especifica ssh como shell remoto

--progress barra de progreso en cada archivo

--stats muestra las estadísticas al final

--exclude excluye un archivo o directorio

Transferencia

scp -r -c arcfour folder/* user@ip:~/dest_folder/

-r recurre a directorios y subdirectorios

-c arcfour opción de cifrado, debe estar habilitada en el sshd
